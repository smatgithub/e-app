# 09 — MSG91 (OTP SMS)

## Purpose
Phone OTP for public login. Pay-as-you-go — keep volume low (OTP only, not order SMS).

## Steps
1. Register [MSG91](https://msg91.com/) + complete KYC.  
2. Register **DLT** templates (India) for OTP.  
3. Create Auth key → `MSG91_AUTH_KEY` in API `.env`.  
4. Sender ID / template ID as required by DLT.

## API env
```
MSG91_AUTH_KEY=
MSG91_OTP_TEMPLATE_ID=
MSG91_SENDER_ID=
```

## Cost control
- Rate-limit `/auth/otp/request`  
- No SMS for order placed/confirmed at Lean (use FCM)  
- Budget alert if SMS > ₹500/mo early

## Staging
Use MSG91 test mode or fixed OTP flag `OTP_STUB=true` (never in Prod).

## Checklist
- [ ] DLT approved  
- [ ] Stub disabled in Prod  
- [ ] OTP works on real phone
